SwagUpload

Github | Site

Web page coupled with an iOS application. Uses Firebase as a lightweight backend. The user can choose an image on their phone via the iOS application and have it uploaded to our server, at which point they can navigate to our web page, enter the secret code the app gives them, and then download the image from the site.

This project was created and developed by myself and two good friends of mine, Eric Zhang and Gates Zeng, at a school-wide "Hack Day" put on by our school, University of California, San Diego. The project is currently unfinished, however enough of it works that we were able to put on a demo for the judges and win second place in the competition!

AutoClicker 2.0

Github

This was my first project ever! I created this back in Spring of 2014, when I was a junior in high school and taking AP computer science. I enjoyed CS so much that, after having learned a decent amount of material, I decided to create a program to help me beat video games!

The inspiration behind this project was a flash game on the internet, where you basically just click a whole bunch of times to win the game. Kind of lame, I know, but its fun when you have a program that can click up to 100,000 times per second! I eventually got tired of clicking on my own, so I decided to write a program to do the clicking for me. I'm still a little proud of my younger self to this day for utilizing that problem-solving mentality.
